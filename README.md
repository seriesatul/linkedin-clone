# LinkedIn Clone - Full Stack Social Media App

This is a full-stack web application built as an internship assignment for Appdost. It's a simplified clone of LinkedIn where users can sign up, log in, create posts, and interact with a public feed.

**Live Project Demo:** [[ Live Vercel Project Link Here]](https://linkedin-clone-beryl-nine.vercel.app/)

---

## Features

- **User Authentication:** Secure user registration and login system using JSON Web Tokens (JWT) for session management.
- **Create Posts:** Logged-in users can create and share text-based posts with the community.
- **Public Feed:** A real-time feed displays all posts from all users, with the newest posts appearing first.
- **Likes & Comments:** Users can engage with posts by liking/unliking and adding comments.
- **Responsive UI:** The user interface is built with Tailwind CSS for a clean, modern, and fully responsive experience on all devices, from mobile phones to desktops.

## Tech Stack

- **Frontend:** React.js (with Vite), React Router, Axios, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (with Mongoose)
- **Authentication:** JSON Web Tokens (JWT), bcrypt.js for password hashing
- **Deployment:**
  - Frontend deployed on **Vercel**.
  - Backend deployed on **Render**.
